# FIREFANGS Esports Website

This is the official website for the FIREFANGS esports team.

## How to Run
Simply open `index.html` in any modern web browser.

## Features
- **Responsive Design**: Works on mobile and desktop.
- **Fire Theme**: Custom animations and particle effects.
- **Roster Showcase**: Cards for each team member.
- **Interactive Elements**: Hover effects, smooth scrolling, and mobile navigation.
